To run tool it is necessary:

  • Define the complete path in which the main files of DeepGravity are present.
  • Define the complete path in which the "plots.ipynb" files are present.
  • Define, in case you won't to use the standard values of the variables, the values to give to the variables in input.
All these operations are possible by using admin area. It is possible to log into the admin area using as credentials for password and username the key “admin”. Once in, clicking on "Cmd commandss ". It is possible to modify all the variables useful for the running of the algorithm. For further information on the variables click here.

If you have correctly done the previous steps, it is possible to run the experiment. To plot the results click here.
