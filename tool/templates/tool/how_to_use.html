Per eseguire il tool è necessario:

  • definire il path assoluto in cui sono presenti i main file di Deep Gravity
  • definire il path assoluto in cui è presente il file "plots.ipynb"
  • definire, nel caso in cui non si vogliano usare i valori standard delle variabili, i valori da dare alle variabili in input
Tutte queste operazioni sono possibili mediante admin area. è possibile accedere all'admin area usando come credenziale per passowrd e username la key "admin". Entrati, cliccando su "Cmd commandss" sarà possibile modificare tutte le variabili utili per l'esecuzione dell'algoritmo. Per maggiori informazioni sulle variabili click here.

Nel caso in cui siano stati eseguiti correttamente i passaggi precedenti, è possibile possibile run the experiment. To plot the results click here.
