Variables for the complete path of the files:

  • main_path: complete Path for the main files of DeepGravity
  • plots_path: complete Path for the files "plots.ipynb" of DeepGravity

Inputs of the algorithm:

  • batch_size: Input batch size for training (default: 1)
  • test_batch_size: Input batch size for testing (default: 1)
  • epochs: Number of epochs to train (default: 10)
  • lr: Learning rate (default: 5e-6)
  • momentum: SGD momentum (default: 0.9)
  • seed: Random seed (default: 1234)
  • log_interval: How many batches to wait before logging training status(default: 1)
  • device: Wheter this is running on cpu or gpu (default: cpu)
  • mode: Can be train or test (default: train)
  • tessellation_area: The area to tessel if a tessellation is not provided (default: United Kingdom)
  • tessellation_size: The tessellation size (meters) if a tessellation is not provided (default: 25000)
  • tile_id_column: Column name of tile's identifier (default: tile_ID)
  • tile_geometry: Column name of tile's geometry (default: geometry)
  • oa_id_column: Column name of oa's identifier (default: oa_ID)
  • oa_geometry: Column name of oa's geometry (default: geometry)
  • flow_origin_column: Column name of flows' origin (default: origin)
  • flow_destination_column: Column name of flows' destination (default: destination)
  • flow_flows_column: Column name of flows' actual value (default: value)

To notice: While it is possible to not modify the input, leaving it with its own default values, it is absolutely requested the modification of main_path and plots_path for the correct run of the algorithm.
