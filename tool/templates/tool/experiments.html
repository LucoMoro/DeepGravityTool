It is finally possible to run the algorithm. For how it has been programmed, DeepGravity allows not only to run DG, but also allows to run the other three algorithms: NG, MFG e G, briefly explained here. Their execution is useful for the comparison of the results produced by DeepGravity with other algorithms, capable of working on similar dataset:

  • run DG
  • run G
  • run NG
  • run MFG

In case you want to run all four of the algorithms it is possible by clicking here.

To notice: Running an algorithm, the page of the occured success will be uploaded only at the end of the run of the same. So it is normal that the page takes time to upload. It is also not recommended, even if possible, to run different operations whilst waiting for the page to run the algorithm. This infact could lead to an error during the run phase of the algorithm.
